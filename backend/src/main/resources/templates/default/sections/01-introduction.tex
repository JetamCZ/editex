\section{Introduction}

Welcome to your new LaTeX project! This template demonstrates the basic features and structure of a LaTeX document.

\subsection{What is LaTeX?}

LaTeX is a typesetting system widely used for producing scientific and mathematical documents. It provides:

\begin{itemize}
    \item High-quality typesetting
    \item Automatic numbering and cross-referencing
    \item Professional mathematical formulas
    \item Consistent formatting throughout documents
\end{itemize}

\subsection{Document Structure}

This document is organized into multiple files:

\begin{enumerate}
    \item \texttt{main.tex} -- The main document file with preamble
    \item \texttt{sections/01-introduction.tex} -- This introduction
    \item \texttt{sections/02-examples.tex} -- Examples of LaTeX features
\end{enumerate}

Using multiple files helps keep your project organized and makes collaboration easier.
