\section{LaTeX Features Examples}

This section demonstrates various LaTeX features you can use in your documents.

\subsection{Mathematical Equations}

Inline math: The quadratic formula is $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.

Display math with equation numbering:
\begin{equation}
    E = mc^2
\end{equation}

Multiple aligned equations:
\begin{align}
    f(x) &= x^2 + 2x + 1 \\
    f'(x) &= 2x + 2 \\
    f''(x) &= 2
\end{align}
